# 🎻 오케스트라 키오스크 출결 시스템

이 프로젝트는 학교 오케스트라 연습실 입구에 설치하여 학생들이 스스로 출결을 남길 수 있도록 돕는 **키오스크형 관리 도구**입니다. 모든 데이터는 각 선생님의 **구글 스프레드시트**에 실시간으로 저장됩니다.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/deploy)

---

## ✨ 주요 기능
- **키오스크 모드**: 대형 입력창을 통한 학번/QR코드 입·퇴실 확인.
- **실시간 현황**: 현재 연습 중인 인원 및 오늘 총 참여 인원 실시간 표시.
- **관리자 대시보드**: 학생 명단 관리 및 월별 누적 연습 시간 통계 제공.
- **초간편 연동**: 구글 클라우드 가입 없이 Apps Script 주소만으로 즉시 연동.

---

## 🚀 3분 만에 나만의 앱 배포하기 (다른 선생님용)

이 앱을 본인의 학교에서 사용하고 싶은 선생님들은 아래 순서대로 따라 하세요.

1.  **코드 복사**: 이 저장소(Repository)를 본인의 GitHub 계정으로 **Fork** 하거나 파일을 다운로드하여 업로드합니다.
2.  **구글 시트 설정**: `초간편_연동_가이이드.md` 파일을 읽고 본인의 구글 시트에 Apps Script를 설치합니다.
3.  **배포하기**: 상단의 **[Open in Streamlit]** 버튼을 눌러 본인의 GitHub 계정을 연결하고 배포합니다.
4.  **연결**: 배포된 앱의 사이드바에 본인의 **Apps Script 웹 앱 URL**을 붙여넣으면 끝!

---

## 📂 파일 구성
- `app.py`: 스트림릿 메인 프로그램 구성 파일.
- `requirements.txt`: 앱 구동에 필요한 라이브러리 목록.
- `구글_스크립트_코드.gs`: 구글 시트에 붙여넣을 서버 코드.
- `초간편_연동_가이이드.md`: 전체 설정 과정 상세 안내서.
