# 🎻 오케스트라 통합 관리 시스템 (출결 + AI 피드백)

이 프로젝트는 학교 오케스트라 운영에 필요한 **출결 관리**와 **AI 연습 피드백**을 하나로 통합한 올인원 솔루션입니다.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/deploy)

---

## ✨ 핵심 기능
1. **출결 체크 (키오스크)**: 대형 화면 UI를 통해 학생들이 직접 학번/QR로 간편하게 입·퇴실 기록.
2. **AI 연습 피드백**: 모범 연주와 학생 연주를 비교하여 피치(음정), 템포(박자), 다이나믹을 AI로 분석하고 점수 및 리포트 제공.
3. **관리자 모드**: 학생 명단 관리 및 연습 데이터 확인.
4. **초간편 연동**: 구글 시트 주소 하나로 즉시 데이터베이스 구축 (GCP 가입 불필요).

---

## 🚀 3분 만에 나만의 통합 시스템 배포하기

1.  **코드 복사**: 이 저장소를 본인의 GitHub 계정으로 업로드합니다.
2.  **구글 시트 설정**: `초간편_연동_가이이드.md` 파일을 따라 출결용 구글 시트를 세팅합니다.
3.  **배포하기**: 상단의 **[Open in Streamlit]** 버튼을 눌러 본인의 GitHub에 연결하고 배포합니다.
4.  **사용**: 
    - 출결 관리를 위해 구글 시트 URL을 입력합니다.
    - 연습 피드백 탭에서 학생들의 연주 음원을 분석합니다.

---

## 📂 파일 구성
- `app.py`: 통합 관리 시스템 메인 앱.
- `analyzer.py`, `visualizer.py`, `feedback.py`: AI 음원 분석 엔진.
- `requirements.txt`: 필요한 라이브러리 목록 (librosa, plotly 등 추가됨).
- `구글_스크립트_코드.gs`: 출결 관리용 구글 시트 서버 코드.
