# 🎻 초간편 구글 시트 연동 가이드 (딱 3분컷!)

이 방식은 복잡한 구글 클라우드 가입 없이, 구글 시트 주소 하나로 앱을 바로 연결하는 가장 쉬운 방법입니다.

---

### 1단계: 구글 시트 준비 (30초)
1. 구글 시트를 새로 만드세요.
2. 하단 탭 이름을 각각 **`명단`**과 **`로그`**로 바꿉니다.
3. 각 시트의 1행(헤더)에 다음 내용을 적습니다.
   - **`명단` 시트**: 이름, 학번, 악기군, QR코드ID
   - **`로그` 시트**: 날짜, 학번, 이름, 입실시간, 퇴실시간, 연습시간(분)
4. 우측 상단 **[공유]** 버튼 -> 일반 액세스를 **[링크가 있는 모든 사용자]**, 권한은 **[편집자]**로 변경합니다.

### 2단계: 스크립트 복사 및 배포 (1분)
1. 구글 시트 상단 메뉴 [확장 프로그램] -> **[Apps Script]**를 클릭합니다.
2. 열린 창에 있는 기존 코드를 다 지우고, 폴더 안의 **`구글_스크립트_코드.gs`** 내용을 통째로 복사해서 붙여넣습니다.
3. 상단 **[배포]** -> **[새 배포]**를 클릭합니다.
4. 유형 선택(톱니바퀴) -> **[웹 앱]** 선택.
   - 설명: `Orchestra API` (아무거나)
   - 다음 사용자로 실행: **[나]**
   - 액세스 권한이 있는 사용자: **[모든 사용자]** (매우 중요!)
5. **[배포]**를 누르고, 중간에 뜨는 [권한 검토]를 눌러 본인 계정을 선택해 허용해 줍니다.
6. 배포 완료 후 화면에 뜨는 **[웹 앱 URL]**을 복사합니다.

### 3단계: 앱에 입력 (30초)
1. 우리 출결 앱을 실행합니다.
2. 사이드바 설정창에 방금 복사한 **웹 앱 URL**을 붙여넣습니다.
3. 즉시 명단이 로딩되며 키오스크 모드가 활성화됩니다!

---

### ✅ 이제 끝입니다!
이 주소만 있으면 다른 선생님들도 본인의 구글 시트에 즉시 연결해서 쓸 수 있습니다.
설정창에 URL을 한 번만 입력해두면 브라우저가 기억하므로 다음부터는 바로 본인 데이터가 뜹니다.
