# 🌐 내 앱을 인터넷에 무료 배포하기 (Streamlit Cloud)

이 과정을 마치면 전용 웹 주소(예: `my-orchestra.streamlit.app`)가 생기며, 누구나 그 주소로 접속해 사용할 수 있습니다.

---

### 1단계: 깃허브(GitHub)에 코드 올리기
1. [GitHub](https://github.com/)에 로그인하고 **[New repository]**를 만듭니다.
   - 이름 예시: `orchestra-attendance`
   - **Public**으로 설정하는 것이 좋습니다 (무료 배포용).
2. 현재 컴퓨터의 `8orchestra-manager/attendance-app/` 폴더 안에 있는 파일들을 모두 업로드합니다.
   - **필수 파일**: `app.py`, `requirements.txt`
   - (가이드 파일들도 함께 올려두면 다른 선생님들이 보기 좋습니다.)

### 2단계: Streamlit Cloud 연결
1. [Streamlit Cloud](https://share.streamlit.io/)에 접속하여 GitHub 계정으로 로그인합니다.
2. **[Create app]** -> **[Yup, I have an app]**을 클릭합니다.
3. 방금 만든 **Repository**를 선택합니다.
4. **Main file path**를 `app.py`로 설정합니다.
5. **[Deploy!]** 버튼을 누릅니다.

### 3단계: 배포 완료 및 주소 공유
1. 약 1~2분 정도 기다리면 나만의 웹 주소가 생성됩니다.
2. 이제 그 주소를 다른 선생님들께 카톡이나 문자로 공유하면 끝!

---

### 💡 다른 선생님들께 알려줄 내용
다른 선생님들께 앱 주소를 주실 때 다음 멘트를 함께 보내주세요:

> "이 주소로 들어오셔서 사이드바 설정창을 확인해 보세요! 가이드를 따라 본인 구글 시트 주소를 연결하면 바로 개인 출결 시스템으로 사용하실 수 있습니다."

---

### ⚠️ 주의사항
*   로컬에서 테스트하던 `.streamlit/secrets.toml`이나 `credentials.json` 같은 파일은 **절대 깃허브에 올리지 마세요.** (지금 새 버전에서는 이 파일들이 필요 없으니 폴더에서 지우고 올리시면 안전합니다.)
